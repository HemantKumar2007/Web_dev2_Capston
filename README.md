## 📌 Project Overview

AI Productivity Dashboard is a React-based web application that helps users manage their daily tasks, notes, calendar events, and get AI-powered smart suggestions — all in one place with a modern dark gold premium UI.

This project was built as Assignment #3 — Frontend Project using React.js with proper component-based architecture, global state management using Context API, and a fully custom CSS theme.

---

## 🚀 Features

- ✅ **Task Manager** — Add, complete, delete and filter tasks by priority (High, Medium, Low)
- 📝 **Notes** — Create colorful sticky notes with 6 custom color options
- 📅 **Calendar** — Monthly calendar view with color-coded upcoming events
- 🤖 **AI Assistant** — Mock smart suggestions with dismiss feature and interactive AI chatbot
- 🌤️ **Weather Widget** — Simulated weather data with 4-day forecast
- ⏰ **Reminders** — Track urgent and upcoming reminders
- 📊 **Dashboard** — Overview with stats cards, daily progress bar and AI insights
